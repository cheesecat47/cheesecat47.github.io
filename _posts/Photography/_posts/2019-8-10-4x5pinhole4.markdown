This is my first Arduino project.

There are 4 parts.
* 1 Arduino Uno
* 1 LCD
* 1 Servo
* 1 Joystick

And there are 6 menu.
1. Set Seconds.
2. Set Minutes.
3. Set Hours.
4. Confirm the time.
5. Set timer.
6. Run Shutter.
